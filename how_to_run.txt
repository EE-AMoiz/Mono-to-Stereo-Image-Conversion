Step1:
-----
Clone and run the DepthPro model:
https://github.com/apple/ml-depth-pro.git

run:
depth-pro-run -i "..." -o "..." --skip-display

Step2:
-----
Run the R_MASK.py or L_MASK.py script to get the translated views.

python R_MASK.py --orig '...' --disp '...' --output '...' \
  --scale-factor 6 --translation 1

Step3:
-----
Clone the Repository from:
https://github.com/lorenzo-stacchio/Stable-Diffusion-Inpaint.git

(Replace the inpaint_inference.py with our inpaint_inference.py. Use the model given in the checkpoints folder.)

Run the model:
python inpaint_inference.py --indir ./dataset/mask --outdir ./dataset/image \
  --ckpt logs/checkpoints/right.ckpt \
  --yaml_profile configs/latent-diffusion/inpainting_example_overfit.yaml \
  --device cuda --steps 50 --prefix finetuned --ema

Step4:
Run align_replace_edgesafe.py to improve the quality.

Step5:
Run align_side_by_side.py to join the left and right views together to be able to view in the headset.
